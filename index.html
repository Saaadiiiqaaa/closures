1. COUNTER

            function counter(){
                var counter = 0;
                function IncreaseCounter() {
                console.log(counter += 1);
                
                }
                return IncreaseCounter;
               }
               var counter = counter();
               counter();
               counter();
               counter();
               counter();

OUTPUT

1
2
3
4

Explaination:-- A closure is a function having access to the parent scope,
even after the parent function has closed. here the function IncreaseCounter has acces to the parent variable
and the variable is protected by the parent function and only the IncreaseCounter function can make changes to that counter varaible
the selfinvoking function IncreaseCounter function reset the value of counter each time its called
